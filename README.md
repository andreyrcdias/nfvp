## Next-frame video prediction

Next-frame video prediction using Conv LSTM and FC-LSTM.


## Quick start

1. Create a virtual environment:
```bash
python -m venv .venv
```

2. Activate it:
```bash
source .venv/bin/activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run:
```bash
jupyter notebook
```
