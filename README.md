## Quick start

1. Create a [virtual environment](https://docs.python.org/3/library/venv.html):
```bash
python -m venv .venv
```

2. Activate it:
```bash
source .venv/bin/activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run:
```bash
jupyter notebook
```
